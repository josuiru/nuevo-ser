# Créditos de las atmósferas fotográficas

Cada fichero JPG/PNG de esta carpeta lleva una entrada aquí con su
autor, URL fuente, licencia y fecha de descarga. Cuando se distribuya
una build del juego con estas fotos embebidas, este fichero forma
parte del paquete redistribuido (los créditos viajan con el contenido).

La pantalla in-app `PantallaCreditos` reproduce este inventario con
formato leíble desde dentro del juego — la atribución visible al
usuario es requisito de las licencias CC-BY y CC-BY-SA.

## Política

Sólo se aceptan licencias compatibles con CC-BY-SA 4.0 (la licencia
del contenido del juego):

- **Dominio público** — sin atribución obligatoria, encaja
- **CC0** — sin atribución obligatoria, encaja
- **CC-BY** (cualquier versión) — atribución obligatoria, encaja
- **CC-BY-SA** (cualquier versión) — atribución obligatoria + ShareAlike, encaja

**NO** se aceptan: NC (non-commercial), ND (no-derivatives) ni
licencias propietarias. Si una imagen es perfecta pero la licencia
no es compatible, no se incluye.

## Procesado

Cada foto se descargó del original de Wikimedia Commons, se
redimensionó a 1920px de lado mayor con `convert` (ImageMagick,
calidad JPEG 80, EXIF eliminado) y se guardó con el id del ambiente
del juego como nombre. El total del bundle son ~7 MB para 11 fotos.
